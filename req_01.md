1. 프로젝트 개요 
- image 도구 프로그램을 개발해야 함. 예를 들어 포토샵의 lite 버전 같은 거야.
- 또는 picsart 프로그램의 일부 기능을 구현해야돼.
- 해당 프로그램의 요구 기능들은 이미 시중에 있는 기능들이고 구현이 크게 어렵지 않을거라 판단됨

2. 기술 스택
- Windows 환경
- python 3
- 이외에 필요한 기술은 적절한 것으로 너가 정하면돼. 

3. 요구 사항
- 이미지 업로드 기능
- 이미지 크기 조절 기능
- 품질 개선 기능
- 이미지에 텍스트 추가 기능
- 보정 기능
- 배경 제거 기능 (피사체 만 남도록)
- 프린트 출력 기능
- 이미지 다운로드 기능
- 프로그램의 화면이 최신 유행하는 느낌으로 이쁘게 나왔으면 좋겠어.
